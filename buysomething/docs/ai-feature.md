## 一、概述

本文档说明项目中的 AI 智能客服功能实现，包括所采用的模型、核心功能以及技术架构。

## 二、采用的 AI 模型

### 2.1 模型选择

| 项目 | 说明 |
|------|------|
| **模型名称** | DeepSeek Chat |
| **模型标识** | `deepseek-chat` |
| **API 版本** | DeepSeek V3 |
| **调用方式** | RESTful API (HTTPS) |
| **API 端点** | `https://api.deepseek.com/chat/completions` |


### 2.2 模型参数配置

```javascript
{
  model: 'deepseek-chat',
  temperature: 0.7,      // 创造性平衡
  max_tokens: 800        // 单次回复最大字数
}
```

| 参数 | 值 | 说明 |
|------|-----|------|
| temperature | 0.7 | 控制回复随机性，0.7 兼顾准确与创意 |
| max_tokens | 800 | 限制回复长度，避免资源浪费 |

## 三、实现的功能

### 3.1 智能客服对话

**功能描述**：用户可通过聊天界面与 AI 客服进行实时对话，获得商品相关问题的智能回答。

**实现文件**：
- `cloudfunctions/ai-chat/index.js` - 云函数（后端）
- `pages/chat/chat.js` - 聊天页面逻辑
- `pages/chat/chat.wxml` - 聊天页面结构
- `pages/chat/chat.wxss` - 聊天页面样式

**核心流程**：
```
用户输入问题 
    ↓
前端发送请求 → wx.cloud.callFunction({ name: 'ai-chat' })
    ↓
云函数接收请求，构建 Prompt
    ↓
调用 DeepSeek API 获取回答
    ↓
返回回答至前端，展示对话
```

### 3.2 商品上下文感知

**功能描述**：当用户从商品详情页进入客服时，AI 自动获取商品信息，提供更具针对性的回答。

**携带信息**：
| 字段 | 说明 |
|------|------|
| name | 商品名称 |
| price | 商品价格 |
| subtitle | 商品描述 |
| colors | 可选颜色 |
| sizes | 可选尺寸 |
| sales | 销量 |
| rating | 评分 |
| stock | 库存 |

**应用场景**：
- 用户咨询"这件衣服怎么洗？" → AI 结合商品类型给出洗涤建议
- 用户咨询"160cm 穿什么码？" → AI 根据商品尺码表给出建议
- 用户咨询"质量怎么样？" → AI 结合销量和评分回答

### 3.3 对话历史记忆

**功能描述**：AI 能够记忆最近 6 轮对话上下文，实现连续对话。

**实现方式**：
```javascript
// 保留最近 6 条对话记录
const recentHistory = history.slice(-6)
```

**优势**：
- 支持多轮追问
- 避免用户重复说明背景
- 对话更自然流畅

### 3.4 快捷问题推荐

**功能描述**：新用户进入客服时，显示"猜你想问"快捷按钮，降低使用门槛。

**预设问题**：
- "这件商品有什么特点？"
- "尺码怎么选？"
- "发什么快递？"

### 3.5 转人工客服入口

**功能描述**：提供"转人工"按钮，复杂问题可转接人工客服处理。
