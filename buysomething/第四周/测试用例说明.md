## 使用方法

### 1. 进入测试页面

在微信开发者工具中：
1. 打开项目
2. 点击底部 tabBar 的「我的」
3. 点击「🧪 API 测试」按钮

### 2. 选择测试类别

- **全部**: 运行所有测试 (29个)
- **后端API**: 只运行API测试 (16个)
- **前端组件**: 只运行前端测试 (9个)
- **Mock测试**: 只运行Mock测试 (4个)

### 3. 运行测试

点击「▶ 开始测试」按钮，等待测试完成

### 4. 查看结果

- **统计栏**: 显示总数、通过数、失败数、通过率
- **结果区**: 显示每个测试的通过/失败状态
- **日志区**: 显示详细执行日志

### 5. 导出报告

点击「📤 导出测试报告」按钮，报告将复制到剪贴板
