# 测试操作指南（学生版）

本文档将教你如何使用两种方案测试云函数 API，并获取所需的截图。

---

## 准备工作

### 1. 打开微信开发者工具

1. 启动微信开发者工具
2. 打开项目：`f:/code/buysomething`
3. 确保项目能正常编译运行

### 2. 检查云函数是否已部署

在微信开发者工具中：
1. 点击「云开发控制台」（左侧云图标）
2. 查看「云函数」列表，确认以下函数已部署：
   - login
   - products
   - product-detail
   - cart
   - orders

---

## 方案一：现有页面测试

### 使用方法

**首页已自动添加测试代码**，当你打开首页时，会自动调用云函数并输出结果到控制台。

### 测试步骤

1. 在微信开发者工具中，点击「编译」
2. 首页加载后，点击「调试器」→「控制台」
3. 查看控制台输出的测试日志

### 截图位置

#### 截图 1：控制台输出（前端 API 调用）

1. 微信开发者工具 → 调试器 → 控制台
2. 找到带有 `【测试】` 前缀的日志
3. 截图保存为 `前端API调用截图.png`

#### 截图 2：应用界面

1. 保持在模拟器界面
2. 点击右上角「截图」按钮
3. 截图保存为 `应用界面截图.png`

---

## 方案二：独立测试页面

### 使用方法

我们已经为你创建了专门的测试页面。

### 测试步骤

1. 在微信开发者工具的地址栏输入：`/pages/test/test`
2. 按回车键进入测试页面
3. 页面会自动开始测试所有云函数
4. 查看测试结果

### 快速访问方法

在任意页面添加按钮跳转到测试页面，或在 `app.json` 中临时添加 tabBar 入口。

### 截图位置

#### 截图 1：测试页面日志

1. 进入测试页面后，等待测试完成
2. 截图页面中的日志区域
3. 保存为 `测试日志截图.png`

#### 截图 2：测试结果

1. 滚动查看测试结果列表
2. 截图保存为 `测试结果截图.png`

#### 截图 3：应用界面

1. 在测试页面的同时，也可以返回首页
2. 截图模拟器界面
3. 保存为 `应用界面截图.png`

## 获取后端 API 响应截图

### 方法一：微信开发者工具 Network

1. 打开调试器 → Network 标签
2. 重新进入测试页面或刷新
3. 找到 `tcbfunc` 开头的请求
4. 点击请求，查看 Response
5. 截图保存为 `后端API响应.png`

### 方法二：CloudBase 控制台

1. 打开 [CloudBase 控制台](https://console.cloud.tencent.com/tcb)
2. 选择环境 → 云函数
3. 选择任意云函数（如 `login`）
4. 点击「日志」标签
5. 查看调用记录
6. 截图保存为 `云函数日志.png`

---

## 获取数据库数据截图

1. 打开 [CloudBase 控制台](https://console.cloud.tencent.com/tcb)
2. 选择环境 → 云数据库
3. 选择集合（如 `products`）
4. 查看数据
5. 截图保存为 `数据库数据截图.png`

## 批量导入测试数据

如果测试时提示"商品不存在"，需要先导入测试数据。

### 方法一：手动添加

1. CloudBase 控制台 → 云数据库 → products 集合
2. 点击「添加记录」
3. 填写商品信息：
```json
{
  "productId": "prod-001",
  "name": "测试商品",
  "price": 99.00,
  "stock": 100,
  "categoryId": "cat-001",
  "image": "https://via.placeholder.com/300",
  "description": "测试商品描述",
  "status": "on_sale"
}
```

### 方法二：批量导入 JSON

1. 准备 JSON 文件：
```json
[
  {
    "productId": "prod-001",
    "name": "iPhone 15 Pro",
    "price": 7999.00,
    "stock": 50,
    "categoryId": "cat-phone",
    "image": "https://via.placeholder.com/300",
    "description": "Apple iPhone 15 Pro",
    "status": "on_sale"
  },
  {
    "productId": "prod-002",
    "name": "AirPods Pro",
    "price": 1999.00,
    "stock": 100,
    "categoryId": "cat-audio",
    "image": "https://via.placeholder.com/300",
    "description": "Apple AirPods Pro",
    "status": "on_sale"
  }
]
```

2. CloudBase 控制台 → 云数据库 → products 集合
3. 点击「导入」
4. 上传 JSON 文件

## 常见问题

### Q: 测试显示"用户不存在"

**解决**：使用测试账号 `13800138001` / `123456`，或先在 CloudBase 控制台的 `users` 集合中添加用户数据。

### Q: 测试显示"商品不存在"

**解决**：在 CloudBase 控制台的 `products` 集合中添加商品数据（见上文批量导入）。

### Q: 云函数调用失败

**检查**：
1. 确认云函数已部署
2. 确认环境 ID 正确
3. 查看 CloudBase 控制台云函数日志
