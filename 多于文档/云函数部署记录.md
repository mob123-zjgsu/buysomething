## 部署详情

### 1. login - 用户登录云函数

```json
{
  "name": "login",
  "runtime": "Nodejs18.15",
  "handler": "index.main",
  "description": "用户登录 HTTP 云函数"
}
```

**功能**:
- 手机号密码登录
- 支持测试验证码 `0000` 跳过验证
- 返回 token 和用户信息

**请求示例**:
```bash
curl -X POST https://buysomething-6gbmbtpxff05be35.service.tcloudbase.com/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800138001","password":"123456","code":"0000"}'
```

## 后续步骤

### 配置云接入（HTTP 触发器）

云函数已部署，但需要配置**云接入**才能通过 HTTP 访问：

1. 登录 [CloudBase 控制台](https://console.cloud.tencent.com/tcb)
2. 选择环境 → **云接入**
3. 创建 HTTP 访问服务
4. 绑定已部署的云函数

**HTTP 访问地址格式**:
```
https://{envId}.service.tcloudbase.com/{functionName}
```

## 响应格式

所有接口返回统一 JSON 格式：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

### 状态码说明

| code | 说明 |
|------|------|
| 0 | 成功 |
| 1001 | 参数错误 |
| 1002 | 未登录/认证失败 |
| 1004 | 资源不存在 |
| 2001 | 服务器错误 |
