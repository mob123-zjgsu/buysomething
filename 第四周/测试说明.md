# API 测试说明

## 测试方式

本项目提供两种测试方式：

---

## 方式一：独立测试页面（推荐）

### 入口
1. 打开微信开发者工具
2. 进入「我的」页面
3. 点击红色的「🧪 API 测试」按钮

### 测试内容

| 步骤 | 测试功能 | 调用云函数 |
|------|----------|-----------|
| 1 | 用户登录 | login |
| 2 | 获取商品列表 | products |
| 3 | 获取商品详情 | product-detail |
| 4 | 添加购物车 | cart (action: add) |
| 5 | 获取购物车列表 | cart (action: list) |
| 6 | 创建订单 | orders (action: create) |
| 7 | 获取订单列表 | orders (action: list) |

### 截图位置

测试结果会实时显示在页面上：
- **上半部分**：测试日志（实时滚动）
- **下半部分**：测试结果列表

## 方式二：首页自动测试

### 入口
直接编译运行小程序，首页加载时自动执行测试

### 位置
- **文件**: `pages/home/home.js`
- **函数**: `testCloudFunctions()`

### 代码位置
```javascript
// pages/home/home.js
onLoad(options) {
  this.loadMoreProducts()
  // ==================== [测试代码开始] ====================
  this.testCloudFunctions()
  // ==================== [测试代码结束] ====================
}
```

**文件**: `pages/home/home.js`

```javascript
async testCloudFunctions() {
  // 测试登录
  const loginRes = await wx.cloud.callFunction({
    name: 'login',
    data: { phone: '13800138001', password: '123456', code: '0000' }
  })
  console.log('【测试】登录结果:', loginRes.result)

  // 测试商品列表
  const productsRes = await wx.cloud.callFunction({
    name: 'products',
    data: { page: 1, pageSize: 10 }
  })
  console.log('【测试】商品列表:', productsRes.result)
}
```

## 常见问题

### Q: 登录显示密码错误
**原因**: 数据库存储的是 MD5 加密后的密码
**解决**: 已修复代码，现在可以正确验证

### Q: 商品详情显示服务器错误
**原因**: 测试时 productId 为空
**解决**: 已修复代码，会先获取商品列表再获取详情

### Q: 创建订单显示参数不完整
**原因**: 缺少 addressId 参数
**解决**: 已添加 addressId 参数
