## 项目概述

这是一个基于微信小程序 + 腾讯云 CloudBase 的购物应用，已实现 RESTful API 接口，可供外部系统调用。

| API | HTTP RESTful API (云接入) |
| 测试工具 | Apifox |

## 云函数部署指南

### 1. 部署 HTTP 云函数

每个 HTTP 云函数包含以下文件：
- `index.js` - 云函数入口
- `package.json` - 依赖配置

**部署方式**:

#### 方式一：使用 CloudBase CLI
```bash
# 安装 CLI
npm install -g @cloudbase/cli

# 登录
tcb login

# 部署云函数
tcb functions:deploy login --path cloudfunctions/login
```

#### 方式二：使用 MCP 工具
```javascript
// 使用 createFunction 工具
await call_integration({
  integrationId: 'tcb',
  toolName: 'createFunction',
  arguments: {
    name: 'login',
    runtime: 'Nodejs12',
    handler: 'index.main',
    codePath: 'cloudfunctions/login'
  }
});
```

### 2. 配置云接入

1. 进入 [CloudBase 控制台](https://console.cloud.tencent.com/tcb)
2. 选择环境 → 云接入
3. 创建 HTTP 访问服务
4. 绑定云函数

### 3. 配置安全规则

在部署前，确保数据库安全规则允许 HTTP 访问。

## 前端开发指南

### 使用 HTTP 客户端

```typescript
import { get, post, put, del } from './api/client';
import { productApi, cartApi, orderApi } from './api/todos';

// 获取商品列表
const result = await productApi.getList({
  page: 1,
  pageSize: 20,
  sort: 'sales'
});

// 添加到购物车
await cartApi.add('product-001', 2, { color: '白色' });

// 创建订单
await orderApi.create('address-001', [
  { productId: 'product-001', quantity: 2 }
]);
```

### 使用 Mock 数据

在 `client.ts` 中设置：
```typescript
CONFIG.useMock = true;  // 启用 Mock
CONFIG.useMock = false; // 禁用 Mock，使用真实 API
```
