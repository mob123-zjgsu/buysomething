## 关键代码说明

### 前端 API 调用
```javascript
// 使用 wx.cloud.callFunction 调用云函数
wx.cloud.callFunction({
  name: '云函数名',
  data: { 参数 }
})
```

### 后端云函数格式
```javascript
// cloudfunctions/xxx/index.js
const cloud = require('wx-server-sdk')
cloud.init({ env: 'buysomething-6gbmbtpxff05be35' })

exports.main = async (event, context) => {
  // 小程序调用时 event 直接是参数
  // 返回 { code: 0, message: 'success', data: {...} }
  return { code: 0, message: 'success', data: {} }
}
```

### 密码存储
- 数据库存储 MD5 加密后的密码
- 登录验证时需要将输入密码转为 MD5 比对
