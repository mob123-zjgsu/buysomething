## 测试步骤

### 步骤 1：打开微信开发者工具

1. 启动微信开发者工具
2. 打开项目：`f:/code/buysomething`
3. 确保 `app.js` 中的环境 ID 正确

### 步骤 2：创建测试页面

在任意页面（如 `pages/home/home.js`）的 `onLoad` 中添加测试代码：

```javascript
Page({
  onLoad() {
    this.testLogin()
  },

  async testLogin() {
    console.log('===== 开始测试 =====')
    
    try {
      // 测试登录
      const loginRes = await wx.cloud.callFunction({
        name: 'login',
        data: {
          phone: '13800138001',
          password: '123456',
          code: '0000'
        }
      })
      console.log('登录结果:', loginRes.result)
      
      // 测试商品列表
      const productsRes = await wx.cloud.callFunction({
        name: 'products',
        data: { page: 1, pageSize: 10 }
      })
      console.log('商品列表:', productsRes.result)
      
      // 测试添加购物车（如果有商品）
      if (productsRes.result?.data?.list?.length > 0) {
        const productId = productsRes.result.data.list[0].productId
        const cartRes = await wx.cloud.callFunction({
          name: 'cart',
          data: {
            userId: 'test-user-001',
            productId: productId,
            quantity: 1
          }
        }, 'POST')
        console.log('添加购物车:', cartRes.result)
      }
      
      console.log('===== 测试完成 =====')
      
    } catch (err) {
      console.error('测试失败:', err)
    }
  }
})
```

### 步骤 3：查看测试结果

1. 在微信开发者工具中，点击「编译」
2. 打开「调试器」→「控制台」
3. 查看 `console.log` 输出的测试结果

## 常见问题

### Q: 返回 "用户不存在"

**原因**：测试用户未在数据库中创建

**解决**：使用已存在的用户，或先通过小程序注册页面注册新用户

---

### Q: 返回 "商品不存在"

**原因**：数据库中没有商品数据

**解决**：需要在 CloudBase 控制台的云数据库中添加商品测试数据

---

### Q: 云函数调用失败

**检查**：
1. 确认云函数已成功部署
2. 确认环境 ID 正确
3. 查看云函数日志排查问题

---

## 查看云函数日志

1. 打开 [CloudBase 控制台](https://console.cloud.tencent.com/tcb)
2. 选择环境 → 云函数
3. 选择对应的云函数
4. 点击「日志」查看调用记录

#### 方式 1：通过小程序页面手动添加数据

**测试用户注册**：
1. 打开小程序的登录/注册页面
2. 输入手机号：`13800138001`
3. 输入密码：`123456`
4. 输入验证码：`0000`（测试验证码）
5. 点击注册/登录

**测试商品数据**：
1. 在 CloudBase 控制台 → 云数据库 → products 集合
2. 点击「添加记录」，手动插入商品数据：
```json
{
```

#### 方式 3：通过小程序测试页面批量测试

创建一个专门的测试页面 `pages/test/test.js`，用于批量测试所有 API：

```javascript
Page({
  data: {
    testResults: []
  },

  onLoad() {
    this.runBatchTest()
  },

  async runBatchTest() {
    const results = []
    
    // 1. 测试登录
    results.push({ name: 'login', status: 'testing' })
    const loginRes = await wx.cloud.callFunction({
      name: 'login',
      data: { phone: '13800138001', password: '123456', code: '0000' }
    })
    results.push({ name: 'login', status: 'done', result: loginRes.result })
    this.setData({ testResults: results })
    
    // 2. 测试获取商品列表
    results.push({ name: 'products', status: 'testing' })
    const productsRes = await wx.cloud.callFunction({
      name: 'products',
      data: { page: 1, pageSize: 10 }
    })
    results.push({ name: 'products', status: 'done', result: productsRes.result })
    this.setData({ testResults: results })
    
    // ... 更多测试
  }
})
```

**wxml 页面显示测试结果**：
```xml
<view class="container">
  <view wx:for="{{testResults}}" wx:key="name" class="result-item">
    <text class="name">{{item.name}}</text>
    <text class="status">{{item.status}}</text>
    <text class="result">{{item.result}}</text>
  </view>
</view>
```
