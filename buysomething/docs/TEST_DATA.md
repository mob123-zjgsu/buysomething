1. 访问 [CloudBase控制台 - 数据库](https://tcb.cloud.tencent.com/dev?envId=buysomething-6gbmbtpxff05be35#/db/doc)
2. 选择 `users` 集合
3. 查看或编辑用户数据

#### 通过云函数访问
```javascript
// 查询所有用户
const db = wx.cloud.database()
const res = await db.collection('users').get()

// 根据手机号查询
const res = await db.collection('users')
  .where({
    phone: '13800138001'
  })
  .get()

// 根据OpenID查询
const res = await db.collection('users')
  .where({
    _openid: 'oK8fU5Iy-test-user-001'
  })
  .get()
```
