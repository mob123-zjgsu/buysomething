# 图片资源说明

本目录需要放置以下图片资源:

## 必需图片

### Logo
- `logo.png` - 应用Logo (120x120px)

### Tab栏图标
- `tab-home.png` - 首页图标 (未选中)
- `tab-home-active.png` - 首页图标 (选中)
- `tab-category.png` - 分类图标 (未选中)
- `tab-category-active.png` - 分类图标 (选中)
- `tab-cart.png` - 购物车图标 (未选中)
- `tab-cart-active.png` - 购物车图标 (选中)
- `tab-profile.png` - 我的图标 (未选中)
- `tab-profile-active.png` - 我的图标 (选中)

### Banner轮播图
- `banner1.jpg` - Banner1 (1920x600px)
- `banner2.jpg` - Banner2 (1920x600px)
- `banner3.jpg` - Banner3 (1920x600px)

### 商品图片
- `product1.jpg` ~ `product8.jpg` - 商品列表图片 (800x800px)

### 商品详情图
- `detail1.jpg` ~ `detail3.jpg` - 商品详情长图 (800x1200px)

### 其他
- `default-avatar.png` - 默认头像 (200x200px)
- `empty-cart.png` - 空购物车图标 (200x200px)

### 分类图片
- `category1.jpg` ~ `category9.jpg` - 分类图标 (400x400px)

## 图片尺寸建议

- Logo: 120x120px
- Tab图标: 48x48px (建议使用PNG格式)
- Banner: 1920x600px 或 750x400px
- 商品图片: 800x800px (正方形)
- 商品详情: 宽度800px,高度自适应
- 头像: 200x200px
- 空状态图标: 200x200px

## 注意事项

1. 建议使用WebP格式以减小文件大小
2. 所有图片建议压缩后使用
3. 可以使用占位图服务临时替换,如: `https://via.placeholder.com/800x800`
4. 实际项目中应将图片上传到CDN
