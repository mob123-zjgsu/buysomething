## 什么是覆盖率配置文件？

覆盖率配置文件是告诉测试框架**如何收集和报告代码覆盖率**的文件。

### 前端覆盖率配置 (jest.config.js)

```javascript
module.exports = {
  collectCoverage: true,           // 收集覆盖率
  coverageDirectory: 'coverage',    // 覆盖率报告目录
  collectCoverageFrom: [          // 哪些文件需要统计覆盖率
    'pages/**/*.{js,wxml,wxss}', // 小程序页面文件
    '!pages/test/**'              // 排除测试页面
  ],
  coverageThreshold: {             // 覆盖率阈值
    global: {
      branches: 50,               // 分支覆盖率 ≥ 50%
      functions: 60,              // 函数覆盖率 ≥ 60%
      lines: 60,                  // 行覆盖率 ≥ 60%
      statements: 60              // 语句覆盖率 ≥ 60%
    }
  }
}
```

### 后端覆盖率配置 (pytest.ini)

```ini
[pytest]
testpaths = tests                # 测试文件目录
python_files = test_*.py         # 测试文件命名规则
python_classes = Test*           # 测试类命名规则
python_functions = test_*         # 测试函数命名规则
addopts = -v --cov=app --cov-report=html --cov-report=term-missing
                                 # -v 详细输出
                                 # --cov=app 统计 app 模块覆盖率
                                 # --cov-report 生成覆盖率报告
```
