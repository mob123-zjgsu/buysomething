### 3. Token 过期时间修复

**修改文件：** `cloudfunctions/login/index.js`

**修复前：**
```javascript
const token = Buffer.from(`${user._id}:${user.phone}`).toString('base64');
```

**修复后：**
```javascript
const tokenData = {
  userId: user._id,
  phone: user.phone,
  exp: Date.now() + 7 * 24 * 60 * 60 * 1000 // 7 天过期
};
const token = Buffer.from(JSON.stringify(tokenData)).toString('base64');
```

### 4. 错误信息泄露修复

**修改文件：** `cloudfunctions/login/index.js`

**修复前：**
```javascript
catch (err) {
  return { data: { error: err.message } }; // ⚠️ 暴露内部错误
}
```

**修复后：**
```javascript
catch (err) {
  console.error('登录失败:', err);
  return { message: '服务器错误', data: null }; // ✅ 不泄露内部信息
}
```

## 三、遇到的问题和解决

### 问题 1：CI 在 GitHub Actions 上报错

**问题描述：** `npm ci` 命令失败，项目找不到 package-lock.json

**解决：** 
```yaml
# 修改前
- name: Install dependencies
  run: npm ci

# 修改后
- name: Install dependencies
  run: npm install
```

### 问题 2：glob 模式在 Linux 不匹配

**问题描述：** `pages/**/*.js` 在 Windows 正常，Linux 上不匹配

**解决：** 
```yaml
# 使用条件检查
- name: Run ESLint on pages
  run: |
    if ls pages/*/*.js 2>/dev/null; then
      eslint pages/*/*.js
    fi
```
