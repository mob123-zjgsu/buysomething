\# 购物微信小程序buysomething



\## 项目简介

这是一个基于微信小程序 + Node.js 的简易购物平台，主要用于学习 Git 协作开发与版本控制。



\## 功能模块

\- 用户模块：注册、登录、个人信息

\- 商品模块：浏览、搜索、详情

\- 购物车模块：添加、删除、结算

\- 订单模块：生成订单、查看订单



\## 技术栈

| 模块 | 技术 |

|------|------|

| 前端 | 微信小程序原生 |（董德负责）

| 后端 | Node.js + Express |（潘圣宇负责）

| 数据库 | MySQL 8.0 |（潘圣宇负责）

| 版本控制 | Git + GitHub |
