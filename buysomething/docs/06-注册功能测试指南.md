# 注册功能测试指南

## 快速测试

### 1. 启动小程序
在微信开发者工具中打开项目 `f:/code/buysomething`

### 2. 验证CloudBase配置
检查 `app.js` 中是否已初始化CloudBase:
```javascript
wx.cloud.init({
  env: 'buysomething-6gbmbtpxff05be35',
  traceUser: true
})
```

### 3. 测试注册流程

#### 步骤1: 进入注册页面
1. 启动小程序
2. 点击"我的"标签页
3. 点击登录/注册按钮
4. 点击"立即注册"

#### 步骤2: 填写注册信息

**必填信息**:
- 手机号: `13800138000` (或任意11位手机号)
- 验证码: `123456` (测试环境固定值)
- 密码: `123456` (6-20位)
- 确认密码: `123456` (与密码一致)
- 勾选用户协议

**界面提示**:
- 页面顶部有橙色提示框:"测试环境: 验证码固定为 123456"
- 点击"获取验证码"后,会显示Toast提示:"请输入验证码 123456"
- 验证码输入框placeholder为:"请输入验证码(123456)"

#### 步骤3: 提交注册
1. 点击"立即注册"按钮
2. 等待注册完成
3. 注册成功后自动返回登录页面

#### 步骤4: 使用新账号登录
1. 在登录页面输入刚注册的手机号
2. 输入设置的密码
3. 点击登录
4. 登录成功后进入个人中心

### 4. 验证注册结果

#### 检查本地存储:
在微信开发者工具调试器中:
```
Storage
├── userInfo
└── isLogin: true
```

#### 检查数据库:
在CloudBase控制台查看 `users` 集合:
- 应该能看到新注册的用户记录
- 包含手机号、OpenID、昵称等信息

## 功能测试用例

### 测试用例1: 正常注册流程

**前置条件**: 未注册的手机号

**测试步骤**:
1. 输入手机号: `13900139000`
2. 点击"获取验证码"
3. 输入验证码: `123456`
4. 输入密码: `test123`
5. 确认密码: `test123`
6. 勾选用户协议
7. 点击"立即注册"

**预期结果**:
- 验证码发送成功,开始60秒倒计时
- 密码强度显示为"弱"
- 注册成功提示
- 自动返回登录页面
- 数据库新增用户记录

### 测试用例2: 手机号格式验证

**测试步骤**:
1. 输入手机号: `12345` (少于11位)
2. 点击"获取验证码"

**预期结果**:
- 提示"请输入正确的手机号"
- 不发送验证码

### 测试用例3: 密码强度检测

**测试步骤**:
1. 输入密码: `123456` (6位纯数字)
2. 观察密码强度提示

**预期结果**:
- 密码强度显示为"弱"
- 强度条显示1格红色

### 测试用例4: 密码不匹配

**测试步骤**:
1. 输入密码: `test123`
2. 输入确认密码: `test456`

**预期结果**:
- "立即注册"按钮不可用(灰色)
- 提示两次密码不一致

### 测试用例5: 重复注册

**前置条件**: 手机号 `13800138000` 已注册

**测试步骤**:
1. 使用已注册手机号 `13800138000`
2. 输入验证码: `123456`
3. 输入密码: `newpass`
4. 点击"立即注册"

**预期结果**:
- 提示"该手机号已注册"
- 注册失败

### 测试用例6: 验证码倒计时

**测试步骤**:
1. 输入手机号: `13900139000`
2. 点击"获取验证码"
3. 观察倒计时

**预期结果**:
- 按钮变为灰色,显示"60s后重试"
- 每秒递减
- 60秒后恢复为"获取验证码"

### 测试用例7: 未勾选用户协议

**测试步骤**:
1. 填写完整注册信息
2. 不勾选用户协议
3. 点击"立即注册"

**预期结果**:
- "立即注册"按钮不可用
- 提示需要同意用户协议

## 边界测试

### 最小密码长度
**测试**: 输入5位密码 `12345`
**预期**: 提示密码过短,不能注册

### 最大密码长度
**测试**: 输入21位密码 `123456789012345678901`
**预期**: 只能输入20位

### 特殊字符密码
**测试**: 输入包含特殊字符的密码 `Test@123!`
**预期**: 密码强度显示为"很强"

### 空表单提交
**测试**: 不填写任何信息直接点击注册
**预期**: 按钮不可用,有表单验证提示

## 性能测试

### 注册响应时间
**测试**: 从点击注册到注册完成
**预期**: < 3秒

### 验证码发送速度
**测试**: 从点击获取验证码到收到提示
**预期**: < 2秒

### 云函数调用次数
**测试**: 完成一次注册流程
**预期**: 调用2次云函数(发送验证码 + 注册)

## 安全测试

### SQL注入测试
**测试**: 在手机号中输入SQL语句
**预期**: 被前端验证拦截,不会发送到后端

### XSS攻击测试
**测试**: 在昵称中输入脚本代码
**预期**: 脚本被转义,不会执行

### 暴力破解防护
**测试**: 连续多次使用错误验证码
**预期**: 应该有限流机制(待实现)

## 已知问题

### 1. 短信验证码
**问题**: 当前使用固定验证码 `123456`
**影响**: 无法测试真实短信发送
**解决**: 生产环境需配置腾讯云短信服务

### 2. 密码加密
**问题**: 密码明文存储
**影响**: 安全风险
**解决**: 后续需要使用bcrypt加密

### 3. 验证码校验
**问题**: 验证码未从数据库验证
**影响**: 安全性较低
**解决**: 完善验证码校验逻辑
