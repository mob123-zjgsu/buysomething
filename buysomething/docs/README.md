## 数据库概览

### 核心集合 (10个)

| 集合 | 用途 | 记录数 |
|-------|------|--------|
| users | 用户信息 | 1000+ |
| products | 商品信息 | 1000+ |
| categories | 商品分类 | 20+ |
| cart | 购物车 | 动态 |
| orders | 订单信息 | 动态 |
| order_items | 订单明细 | 动态 |
| addresses | 收货地址 | 5000+ |
| coupons | 优惠券 | 100+ |
| favorites | 收藏 | 动态 |
| banners | Banner轮播 | 10+ |

### 数据关系

```
users (1) ↔ (N) cart
users (1) ↔ (N) orders
users (1) ↔ (N) favorites
users (1) ↔ (N) addresses
orders (1) ↔ (N) order_items
categories (1) ↔ (N) products
```

## API概览

### 接口分类 (26个核心接口)

| 模块 | 接口数量 | 说明 |
|------|---------|------|
| 用户相关 | 5个 | 登录、信息管理、地址管理 |
| 商品相关 | 6个 | 列表、详情、搜索、分类 |
| 购物车相关 | 5个 | 增删改查、清空 |
| 订单相关 | 6个 | 创建、查询、状态管理 |
| 支付相关 | 2个 | 创建支付、状态查询 |
| 其他 | 2个 | 收藏、Banner |

### 接口规范

- **协议**: HTTPS
- **数据格式**: JSON
- **认证方式**: 微信OpenID
- **请求方式**: 云函数调用

## 开发规范

### 命名规范

- 云函数: `{模块名}/{操作名}`
- 数据库集合: `{业务域}_复数形式`
- 字段名: 小驼峰命名法(camelCase)

### 代码规范

- 使用 ES6+ 语法
- 使用 async/await 处理异步
- 统一错误处理和返回格式

### 安全规范

- 所有数据操作基于用户OpenID
- 敏感信息脱敏返回
- 禁止前端直接操作数据库

## 常见问题

### Q: 如何快速开始?

A: 按照 `CLOUDBASE_SETUP.md` 中的步骤,先连接CloudBase环境,然后参考 `03-API设计文档.md` 开发接口。

### Q: 如何测试API?

A: 使用微信开发者工具的"云开发"控制台,或在代码中直接调用云函数。

### Q: 数据库如何初始化?

A: 参考 `02-数据库设计文档.md` 中的字段定义,在CloudBase控制台手动创建集合,或使用云函数批量创建。
