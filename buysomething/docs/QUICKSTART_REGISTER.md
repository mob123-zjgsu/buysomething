# 快速部署 - 注册功能

## 部署前准备

### 1. 确认CloudBase环境
- 环境ID: `buysomething-6gbmbtpxff05be35`
- 已创建数据库集合: `users`, `sms_codes`

### 2. 打开项目
在微信开发者工具中打开项目: `f:/code/buysomething`

## 部署云函数

### 方式一: 微信开发者工具部署(推荐)

#### 步骤1: 上传register云函数
1. 在左侧目录树找到 `cloudfunctions/register`
2. 右键点击 `register` 文件夹
3. 选择"上传并部署:云端安装依赖"
4. 等待上传完成(约30秒)

#### 步骤2: 上传sendSmsCode云函数
1. 在左侧目录树找到 `cloudfunctions/sendSmsCode`
2. 右键点击 `sendSmsCode` 文件夹
3. 选择"上传并部署:云端安装依赖"
4. 等待上传完成(约30秒)

### 方式二: 命令行部署

```bash
# 进入项目目录
cd f:/code/buysomething

# 部署注册云函数
tcb functions:deploy register

# 部署发送验证码云函数
tcb functions:deploy sendSmsCode
```

## 使用注册功能

### 测试账号
- 手机号: `13800138000` (或任意11位手机号)
- 验证码: `123456` (测试环境固定)
- 密码: `123456` (6-20位)

### 测试流程
1. 启动小程序
2. 进入"我的"页面
3. 点击登录/注册
4. 点击"立即注册"
5. 填写注册信息:
   - 输入手机号(任意11位手机号)
   - 点击"获取验证码"(会提示"请输入验证码 123456")
   - 输入验证码: `123456`
   - 设置密码: `123456`
   - 确认密码: `123456`
   - 勾选用户协议
6. 点击"立即注册"
7. 注册成功后自动返回登录页

### 界面提示
注册页面顶部会有橙色提示框:
> 测试环境: 验证码固定为 123456

验证码输入框的placeholder也会提示:
> 请输入验证码(123456)

## 注意事项

### 测试环境
- 验证码固定为 `123456`
- 不会真实发送短信
- 密码明文存储

### 生产环境
- 需要配置腾讯云短信服务
- 验证码随机生成
- 需要加密存储密码

## 常见问题

### Q: 云函数上传失败?
**A**: 
1. 检查网络连接
2. 确认已登录CloudBase
3. 尝试重新上传

### Q: 注册时提示验证码错误?
**A**: 
1. 测试环境请使用固定验证码 `123456`
2. 确认验证码输入正确

### Q: 注册后无法登录?
**A**: 
1. 确认使用的是注册时的手机号和密码
2. 检查CloudBase环境ID是否正确
3. 查看云函数日志排查问题
