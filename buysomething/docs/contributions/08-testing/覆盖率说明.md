# 覆盖率说明

## 项目概况

- **项目类型**：微信小程序购物商城
- **技术栈**：微信小程序 + CloudBase 云函数 + NoSQL 数据库
- **测试方法**：半自动化测试

---

## 覆盖率配置说明

由于微信小程序使用微信开发者工具内建测试框架，不支持传统的前端覆盖率配置文件（如 Jest 的 `jest.config.js` 或 Vitest 的 `vitest.config.ts`）。

因此采用**功能覆盖率**统计方式替代代码覆盖率。
