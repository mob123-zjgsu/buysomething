## 测试方法

### 1. API 测试方法

使用 `wx.cloud.callFunction` 调用云函数，验证：
- **正常流程**：API 返回正确的成功响应
- **异常流程**：验证错误处理和返回码
- **参数校验**：测试边界条件和参数缺失

```javascript
// 示例：登录成功测试
const res = await wx.cloud.callFunction({
  name: 'login',
  data: { phone: '13800138001', password: '123456' }
})
// 期望：result.code === 0
```

### 2. 前端逻辑测试

在测试页面中验证：
- 表单验证正则表达式
- 状态管理逻辑
- UI 状态展示

```javascript
// 示例：手机号格式验证
const phoneReg = /^1[3-9]\d{9}$/
const isValid = phoneReg.test('13800138001') // true
```

### 3. Mock 测试方法

模拟 API 响应场景，验证前端容错能力：
- 网络延迟处理
- 错误信息展示
- 异常捕获
