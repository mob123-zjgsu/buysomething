## 六、问题排查

### 6.1 云函数调用失败

**可能原因**：
1. 云函数未部署
2. 环境 ID 配置错误
3. 网络问题

**解决方法**：
1. 在微信开发者工具中检查云函数状态
2. 确认 `app.js` 中的环境 ID
3. 查看控制台错误日志

### 6.2 商品数据不显示

**可能原因**：
1. 数据库集合为空
2. 查询条件过滤了数据

**解决方法**：
1. 检查 `products` 集合是否有数据
2. 确认商品 `status` 字段为 1（上架）
